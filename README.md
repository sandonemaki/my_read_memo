# yondeco「自分にとってどのぐらい大切な一冊」なのか、「見える化」するアプリ

## サービス概要

yondecoアプリは、いま読んでいる本が自分にとってどのぐらい大切な本なのか、見える化できる読書アプリです。 </br>
アプリの機能に沿って読書メモをとることで、書評家のように自分だけの本のレビューが完成していきます。 </br>
詳細は[トップページ](https://www.yondeco.site)をご確認ください。
</br>
