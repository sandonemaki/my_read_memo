# yondeco / 「自分にとってどのぐらい大切な一冊」なのか、「見える化」するアプリ

## サービス概要

yondecoアプリは、いま読んでいる本が自分にとってどのぐらい大切な本なのか、見える化できる読書アプリです。 </br>
アプリの機能に沿って読書メモをとることで、書評家のように自分だけの本のレビューが完成していきます。

### ▼ yondecoアプリの特徴

2つの読書体験を以下のように名づけました。</br>
**大切な本と出合うための私的な読書 👉 さらさら読書**</br>
**アウトプットにつながる学びの読書 👉 じっくり読書**

**yondecoアプリの取扱読書範囲**</br>
